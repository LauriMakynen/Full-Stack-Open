    participant browser
    participant server

    Note right of browser: User writes a new note into the input field
    Note right of browser: User clicks "Save"

    Note right of browser: SPA JavaScript prevents default form submit\nand creates a JSON payload

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa\nContent-Type: application/json\n{"content":"<user input>","date":"<timestamp>"}
    activate server
    Note right of server: Server stores the new note
    server-->>browser: HTTP 201 Created\n{"message":"note created"}
    deactivate server

    Note right of browser: SPA updates the notes list in memory
    Note right of browser: SPA re-renders the notes list to the DOM\n(no page reload)
